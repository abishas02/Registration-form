Simple Registration Form 
This is a clean and functional registration form that uses JavaScript to perform real-time validation. It is designed to improve user experience by catching errors before the form is even submitted.

📝 Project Requirements
Name Validation: Ensures the name field is not left empty.

Email Validation: Checks for a valid email format (e.g., name@email.com).

Password Security: Requires a minimum of 6 characters to be valid.

Smart Submit Button: The "Register" button stays disabled until all fields are correctly filled out, preventing invalid submissions.

🛠️ Built With
HTML5 for structure.

CSS3 for styling and error visibility.

JavaScript for real-time input monitoring and DOM manipulation.

🚀 How to Use
Clone the repository.

Open index.html in your browser.

Start typing in the form to see the validation in action!# Registration-form
